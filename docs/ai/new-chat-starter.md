# New Chat Starter

Copy this block at the top of a new chat and fill it in.

```md
Task:
Goal:
Scope:
- In:
- Out:

Files expected to change:

Constraints:
- Security:
- Performance:
- UX / design:
- Compatibility:

Validation:
- Commands to run:
- Manual checks:

Definition of done:

Output format:
- Summary
- Files changed
- Checks run
- Risks / follow-ups
```

## Rules for Better Results
- Keep one chat focused on one task.
- Prefer tasks that can be completed in about 30-90 minutes.
- State exact file paths when possible.
- Explicitly list out-of-scope items to prevent drift.
