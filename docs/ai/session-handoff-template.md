# Session Handoff Template

Use this when ending a long chat so the next chat can start quickly.

```md
Task summary:

What is already done:
- 

Files changed:
- 

Checks run and results:
- 

Open issues / risks:
- 

Next best action:
- 
```

## Handoff Rules
- Keep this under 250-400 words when possible.
- Focus on decisions, not raw logs.
- Include exact file paths.
