# Starter Kit

Simple future-project bootstrap.

## Usage (1 copy + 1 command)

1. Copy the `starter-kit/` folder into your new project root.
2. Run:

```bash
node starter-kit/init-project.mjs --name my-project --repo owner/my-project
```

This will scaffold:

- `AGENTS.md`
- `contracts/agent-contract.json`
- `config/security-release-policy.json`
- `docs/easy-mode.md`
- `docs/future-llm-prompt-template.md`
- `docs/links-ingestion.md`
- `public/llms.txt`
- `scripts/*` (workflow, security, contract, skill-graph, links-ingest, easy-ops)
- `skills-graph/*`

If `package.json` exists, it also adds starter scripts:

- `test:workflow-check`
- `test:agent-contract`
- `test:security:policy`
- `test:security:release-gate`
- `test:skill-graph`
- `test:links-ingest`
- `easy:quick`
- `easy:full`
- `starter:quick`
- `starter:full`
- `starter:links`
- `starter:contract`
- `starter:security`
- `starter:skill-graph`

## Force overwrite

Use `--force` to overwrite existing files generated by the kit:

```bash
node starter-kit/init-project.mjs --name my-project --repo owner/my-project --force
```
